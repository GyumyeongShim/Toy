데이터를 읽어서 테이블을 저장하고
저장된 테이블을 바탕으로 리소스를 만든다
만들어진 리소스에 pool 컴포넌트를 추가하거나 안하거나
만들어진 오브젝트를 스테이지에 넣어서 관리한다
배치된 오브젝트가 담긴 스테이지를 씬에 관리한다
data->resource->effect,pool,ui->stage->scene

모든 씬에는
1. MainCamera, Light, StageController를 갖는다
2. StageMaster는 Manager 관련을 Init()해준다.
3. StageMaster는 Data에 저장된 Stage 구상을 진행해준다

Pool이 해야할 일
어떤 오브젝트? GameObject m_origin
사용 여부 bool m_isUsing
초기화 init()
원본 만들어 두기 Create()


전부 제거하기 Clear()

글자 + pool 갯수로 저장해두기 dictionary<string, Pool> m_dicPool

pool 만들어두기 CreatePool()
pool 빌려주기 GetPool() / DnqueuePool() / PopPool()
pool 반납받기 ReturnPool() /EnqueuePool() / PushPool()
pool 제거하기 RemovePool() /
전부 제거하기 Clear()
